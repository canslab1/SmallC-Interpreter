SmallC-Interpreter

A Small-C Interactive Interpreter for System Software Education

View on GitHub Download ZIP

Overview

SmallC-Interpreter is a fully-featured interactive interpreter for the Small-C programming language, a strict subset of ISO C originally defined by Ron Cain (1980) and James Hendrix (1984). It provides a REPL environment where users can enter Small-C code line by line for immediate execution, or load complete source files for batch execution.

Quick Start

git clone https://github.com/canslab1/SmallC-Interpreter.git
cd SmallC-Interpreter
python3 main.py

Features

  • Interactive REPL with sc> prompt and multi-line input detection
  • Recursive-descent parser with 13-level operator precedence
  • Tree-walking interpreter with simulated byte-addressable memory
  • 23 built-in functions — I/O, string, math, and utility
  • Program buffer management — LOAD, SAVE, LIST, EDIT, DELETE, INSERT, APPEND
  • Execution tracing and state inspection commands
  • Comprehensive error handling with clear messages
